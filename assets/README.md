# Assets

Reusable marketing assets and templates.

## Folder Structure

```
assets/
├── templates/          # GHL funnel/form templates
├── creative/           # Ad images, videos, graphics
├── copy/               # Swipe files, email sequences
└── stock/              # Licensed stock photos
```

## Usage

- Reference assets in campaign plans
- Keep naming consistent: `[type]-[description]-[size]`
- Example: `ad-kitchen-before-after-1080x1080.png`
