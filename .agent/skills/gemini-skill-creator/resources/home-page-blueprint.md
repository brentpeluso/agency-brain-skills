# Remodeler Booked: High-Performance Home Page Blueprint

**Goal**: A localized, lightning-fast landing page that converts Remodeling Contractors into booked strategy calls.
**Aesthetic**: "Sharp Growth" simplicity + "Pinpoint" authority. Clean white space, strong Navy typography, vibrant Green CTAs.

---

## ‚ö° Speed Optimization Rules (CRITICAL)
*   **Images**: ALL images must be converted to **WebP** format and compressed to under 100kb (use [squoosh.app](https://squoosh.app/)).
*   **Video**: Do NOT upload videos to GHL directly. Embed from YouTube/Vimeo/Wistia with "Lazy Load" enabled.
*   **Animations**: Disable GHL "Entry Animations" (Fade in, Slide up) on the Hero section to improve LCP (Largest Contentful Paint).
*   **Fonts**: We are using Google Fonts (Montserrat/Open Sans). In GHL Settings, ensure "Optimize Fonts" is checked if available.
