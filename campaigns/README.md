# Campaigns

Store campaign briefs and strategies here, organized by client or project.

## Folder Structure

```
campaigns/
├── [client-name]/
│   ├── YYYY-MM-DD-campaign-name-brief.md    # From rb-brainstorming
│   └── YYYY-MM-DD-campaign-name-plan.md     # From rb-planning
```

## Naming Convention

- **Client folders:** Lowercase, hyphenated (e.g., `smith-remodeling`, `abc-contracting`)
- **Brief files:** `YYYY-MM-DD-[campaign-name]-brief.md`
- **Plan files:** `YYYY-MM-DD-[campaign-name]-plan.md`

## Example

```
campaigns/
├── smith-remodeling/
│   ├── 2026-01-27-kitchen-promo-brief.md
│   └── 2026-01-27-kitchen-promo-plan.md
└── valley-builders/
    └── 2026-02-01-spring-bath-brief.md
```
